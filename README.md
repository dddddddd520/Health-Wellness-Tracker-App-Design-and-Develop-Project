# CoreWell - Health & Wellness Tracking Platform

CoreWell is a modern web application designed to help users track and manage their health and wellness journey. With an intuitive interface and comprehensive features, it provides a seamless experience for monitoring daily health metrics, scheduling reminders, and receiving personalized insights.

## Features

### 1. Dashboard
- Overview of key health metrics
- Quick access to important features
- Visual representation of health trends
- Personalized health status summary

### 2. Daily Log
- Track daily health metrics
- Record mood and symptoms
- Monitor exercise and nutrition
- Log sleep patterns and quality
- Access to instant medical consultation through "Ask a Doctor" feature

### 3. Insights
- Detailed analysis of health trends
- Personalized recommendations
- Progress tracking
- Health goal monitoring

### 4. Reminders
- Medication reminders
- Appointment scheduling
- Exercise routine alerts
- Hydration and meal timing notifications

### 5. Profile Management
- Personal health information
- Medical history tracking
- Health goals setting
- Emergency contact information

### 6. Doctor Consultation
- Direct access to medical professionals
- Quick question feature
- Chat history
- Professional medical advice

## Technical Stack

- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite (configured for development)
- **Authentication**: Flask-Login
- **UI Framework**: Custom CSS with responsive design

## Installation

1. Clone the repository:
```bash
git clone [repository-url]
cd corewell
```

2. Create a virtual environment and activate it:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python app.py
```

The application will be available at `http://localhost:5000`
